# الرفيق - تطبيق فلاتر

تطبيق محادثات فورية مبني بـ Flutter مع تصميم عصري وحيوي.

## الميزات
- تسجيل دخول وإنشاء حساب
- محادثات فورية عبر WebSocket
- تخزين محلي للرسائل (SQLite)
- تبديل الوضع الليلي/النهاري
- مؤشر الكتابة
- حالة الاتصال (أونلاين/أوفلاين)
- إعادة إرسال الرسائل الفاشلة
- بحث عن مستخدمين

## الإعداد
1. تأكد من تثبيت Flutter SDK (3.0+)
2. غيّر عنوان السيرفر في `lib/config/app_config.dart`
3. شغّل `flutter pub get`
4. شغّل `flutter run`

## هيكل المشروع
```
lib/
├── config/          # إعدادات التطبيق
├── models/          # نماذج البيانات
├── providers/       # إدارة الحالة (Provider)
├── services/        # الخدمات (API, WebSocket, Storage)
├── screens/         # الشاشات
├── widgets/         # الويدجت المخصصة
└── main.dart        # نقطة البداية
```
